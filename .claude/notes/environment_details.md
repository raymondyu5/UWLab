# Environment: UW-FrankaLeap-PourBottle-JointAbs-v0

## Task Overview

Bottle pouring task: Franka arm + LEAP hand must grasp a bourbon bottle and pour it (cap tip) over a pink cup.

**Registration**: `source/uwlab_tasks/uwlab_tasks/manager_based/manipulation/grasp/config/franka_leap/__init__.py`

**Config class**: `PourBottleFrankaLeapJointAbsCfg` (in `tasks/bottle_pour.py`)
- Inherits: `PourBottleFrankaLeapCfg` → `FrankaLeapGraspEnvCfg` → `GraspEnvCfg` → `ManagerBasedRLEnvCfg`

## Physics / Timing

- Physics Hz: 60 Hz (`DEFAULT_GRASP_PHYSICS_HZ`)
- Decimation: 6 (control runs at 10 Hz = 60/6)
- Control Hz: 10 Hz
- Episode horizon: 176 steps (task) + 15 warmup = 191 total
  - Episode length = 191 * 6 * (1/60) ≈ 19.1 seconds

## Scene

- **Robot**: Franka arm (7 DOF) + LEAP hand (16 DOF) = 23 DOF total
- **Table**: USD asset at `assets/table/table_instanceable.usd`, pos [0.3, 0.0, 0.0]
- **Grasp object** (bottle): USD at BOTTLE_USD, spawned at `(0.53, -0.105, 0.11)`
- **Pink cup**: USD at PINK_CUP_USD, spawned at `(0.53, 0.23, 0.07)`
- **Cameras**: In `rl_mode`, cameras are DISABLED (train_camera=None, fixed_camera=None)
  - In `rl_mode`, seg_pc is generated from mesh-based sampling (CachedSamplePC), not camera

## Run Modes

```python
EVAL_MODE = "eval_mode"    # cameras on, mesh-based seg_pc
DISTILL_MODE = "distill_mode"  # cameras on, camera-rendered seg_pc (for distillation)
RL_MODE = "rl_mode"       # cameras OFF, mesh-based seg_pc (fastest)
```
Training command sets `env_cfg.run_mode = "rl_mode"`.

## Warmup Mechanism

After each reset, the env runs `num_warmup_steps=15` steps using `warmup_action()`:
- For JointAbs: holds at reset joint position `ARM_RESET + HAND_RESET`
- This allows physics simulation to settle before returning first obs

RFS also adds `external_warmup = rfs_cfg.num_warmup_steps = 5` extra warmup steps with zero PPO actions.

## Reset Randomizations (Events)

Events run on `mode="reset"`:

### 1. `reset_robot`
- Resets arm to `ARM_RESET = [0.311, 0.004, -0.311, -2.051, 0.001, 2.055, 0.781]` (7D, radians)
- Resets hand to `HAND_RESET` (16D, radians) — specific LEAP hand pose
- Clamps arm joints to `FRANKA_LEAP_ARM_JOINT_LIMITS`

### 2. `reset_object` (bottle)
- Default pos: `(0.53, -0.105, 0.11)` in env-local frame
- Default rot: `BOTTLE_SPAWN_ROT`
- **Pose range** (uniform random):
  - x: ±0.11 m (so bottle x ∈ [0.42, 0.64])
  - y: ±0.095 m (so bottle y ∈ [-0.200, -0.010])
  - z: 0 (no z randomization)
  - roll/pitch/yaw: 0 (no rotation randomization)
- Height respects table block z (for optional table height randomization)

### 3. `reset_cup_object` (pink cup)
- Default pos: `(0.53, 0.23, 0.07)` in env-local frame
- Default rot: `(0.707, 0.707, 0.0, 0.0)` (upright)
- **Pose range**:
  - x: ±0.12 m
  - y: ±0.06 m
  - z: 0
  - roll/pitch/yaw: 0

### 4. `reset_table_block` (optional)
- `table_z_range = (0.0, 0.0)` → no table height randomization (but event still fires)

### 5. `reset_camera` / `reset_fixed_camera`
- **DISABLED in rl_mode**
- When active: randomizes camera position in range `(0.4, -0.15, 0.10, 0.6, 0.15, 0.25, 0.8, 1.7)`, phi ∈ [1.0, 1.66] rad, theta ∈ [0.0, 0.5] rad

### 6. `capture_reset_references`
- Runs AFTER reset_object and reset_cup_object
- Captures bottle height at reset (for reward gating)

## Observation Space

Observations are grouped under `policy` (no concatenation — `concatenate_terms=False`):

| Key | Shape | Description |
|-----|-------|-------------|
| `joint_pos` | (23,) | All robot joint positions |
| `arm_joint_pos` | (7,) | Arm joint positions only |
| `hand_joint_pos` | (16,) | LEAP hand joint positions only |
| `ee_pose` | (7,) | End-effector pose (pos+quat) in world frame |
| `cup_pose` | (3,) | Pink cup XYZ position (env-local) |
| `target_object_pose` | (7,) | Computed target bottle pose for pouring |
| `manipulated_object_pose` | (7,) | Bottle pose (pos+quat, env-local) |
| `seg_pc` | (3, 2048) | Segmented point cloud of robot+bottle+cup |

**Note**: In PPO obs (after stripping), `seg_pc` is removed; `ee_pose` is truncated to 3D.

### Point Cloud (seg_pc)
- Generated by `CachedSamplePC` (mesh-based, no camera needed)
- Samples: 256 pts from arm mesh, 64 pts from hand mesh, N pts from bottle, M pts from cup
- Downsampled to 2048 pts total
- PCD crop region: `[-0.10, -0.50, 0.035, 0.85, 0.50, 0.67]` (real workspace bounds)

## Action Space: JointAbs

**Controller**: `FrankaLeapJointPositionAction`
- `BoundedJointPositionActionCfg`: 23-DOF joint position targets
- Arm joints (7D): clamped to conservative limits `FRANKA_LEAP_ARM_JOINT_LIMITS`
- Hand joints (16D): unclamped absolute joint positions
- Scale: 1.0 (actions in radians)

## Reward Functions

All rewards defined as separate `RewTerm` entries with independent weights:

| Term | Function | Weight | Description |
|------|----------|--------|-------------|
| `grasped` | `pour_grasped` | +1.0 | 1.0 if bottle cap tip z > 0.05m (off table) |
| `xy_healthy` | `pour_xy_healthy` | +2.0 | Dense XY alignment × is_healthy_z gate |
| `xy_near_miss` | `pour_xy_near_miss` | +5.0 | Dense XY alignment × is_near_miss gate |
| `success` | `pour_success` | +10.0 | Sparse success |
| `cup_topple` | `pour_cup_topple` | -10.0 | Penalty if cup tilts >0.524 rad |
| `joint_vel` | `pour_joint_vel_l2` | -1e-3 | L2 penalty on joint velocities |
| `joint_limit` | `pour_joint_pos_limits` | -6e-1 | Penalty for approaching joint limits |
| `action_rate` | `pour_action_rate_l2` | -5e-3 | Penalty for action magnitude changes |

### Key Metrics

- **Bottle cap offset**: `(-0.132179, 0.0, 0.0)` in bottle local frame (cap at X=-13.2cm)
- **is_grasped**: cap tip z > 0.05 m
- **is_healthy_z**: cap tip z ∈ [cup_z + 0.17, cup_z + 0.30]
- **is_near_miss**: XY dist from cap to cup center < 0.15 m AND is_healthy_z
- **is_success**: XY dist < 0.04 m AND is_healthy_z
- **Max XY dist** (for reward normalization): 0.60 m

## Termination Conditions

| Condition | Reason |
|-----------|--------|
| `time_out` | Episode length exceeded (176 steps) |
| `bottle_too_far` | Bottle XY distance > 1.0 m from spawn |
| `cup_toppled` | Cup tilt angle > 0.524 rad (~30°) from upright |

Note: `bottle_dropped` termination exists but is commented out.
